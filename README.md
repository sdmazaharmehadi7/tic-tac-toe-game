
🎮 Tic Tac Toe – Python Console Game

-->A simple Tic Tac Toe game built using Python. This is a console-based application that allows two players to play turn by turn directly in the terminal.
The project focuses purely on game logic implementation without any GUI or web interface.

🚀 Features

-->Two-player mode (X and O)

-->Turn-based gameplay

-->Input validation

-->Automatic winner detection

-->Draw condition handling

-->Simple and clean terminal interface

🛠️ Tech Stack
-->Python (Core concepts only)

📚 What I Learned

-->Conditional statements and loops

-->Function implementation

-->List handling

-->Game logic design

-->Error handling and input validation

▶️ How to Run

-->Clone the repository:

git clone https://github.com/your-username/tic-tac-toe.git

-->Navigate to the project folder:

cd tic-tac-toe

-->Run the program:

python main.py

💡 This project helped me strengthen my Python fundamentals and logical thinking skills.
